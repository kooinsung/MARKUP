글로벌 컨벤션 기업 리젠시컴의 모바일 웹페이지 제작 퀘스트가 오픈 되었습니다.
국내외 컨벤션 행사의 초청, 등록, 참가자 관리 등을 효율적으로 진행하는
간단한 모바일 웹페이지인 ‘리젠시 GATE’ 를 제작하는 IT 관련 팀을 위한 실제 프로젝트입니다.

When / 모집 및 프로젝트 기간

  • 모집기간 : 2015년 5월 3일 일요일 오후 6시
  • 프로젝트 기간 : 2015년 5월 (1개월 이내)

What / 퀘스트 미션

  • 리젠시 GATE 반응형 모바일 웹페이지 기획 및 개발
  • 기본 백오피스 관리자 기능(모바일웹 및 관리자 기능 포함 10페이지 이내)

Who / 참가자격

  • 모바일웹 개발 경험을 보유한 청년 프로젝트 팀
  • 등록, 참가자 관리 관련 솔루션을 보유한 프로젝트 팀 우대

Rewards / 퀘스트 보상

  • 프로젝트 비용 500만원
  • 리젠시컴의 후속 프로젝트 기회

How /퀘스트 진행절차

  • 프로젝트 팀 모집 : 2015년 5월 3일
  • 프로젝트 팀 면접 : 2015년 5월 6일 – 8일
  • 프로젝트 수행 팀 선정 : 2015년 5월 9일
  • 프로젝트 진행 : 2015년 5월 말까지 완료

Apply / 참가방법

  • 팀스퀘어(www.teamsquare.com)에서 ‘리젠시 GATE’ 퀘스트 페이지
  • 제출서류 - 팀 소개 및 포트폴리오 (자유양식 / 팀원 및 경력위주)
