메뉴 설정

Mobile 서비스의 메인 메뉴를 관리 합니다.

이용안내
  • App 새 메뉴 만들기: 또는 좌측상단 버튼 클릭
  • 현재 APP 메뉴 수정: 좌측에서 수정 할 메뉴명 클릭
  • 현재 APP 메뉴 순서 수정: 좌측에서 수정할 메뉴명 클릭 후 상단 버튼 클릭
  • APP 메뉴 미사용 하기: 버튼 클릭
  • 미사용 메뉴 사용전환 하기: 미사용 상태에서 좌측 메뉴명 클릭 후 버튼 클릭

※ 삭제는 수정페이지에서 삭제 가능합니다. 삭제 후 복원 되지 않으므로 신중히 처리 바랍니다.
※ 노출을 원하지 않으실 경우 메뉴 선택 후 “미사용” 선택하십시오.
